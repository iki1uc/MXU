# MXU Root System

## PURPOSE
MXU ist das Multi‑X‑Unit System mit Achsen AX, IS, XA und Moves MX, UX, IU.

## FILES
- index.html → PX Receiver
- go.html → AXINXA + MXU Channel
- ch.html → Instanzen (root/axis/room)
- mxu.js → PX Pipeline
- AXINXA.js → Achsen + Moves
- respo.js → Text‑Analyse
- nc.evo.js → EVO‑Analyse

## PIPELINE
PX → respo → evo → AXINXA → MXU Channel

## NOTES
